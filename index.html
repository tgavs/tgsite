IMSS satisfaction analysis by Tomas Gaviño

Has the IMSS Satisfaction Improved?

Every quarter the Mexican Healthcare Institute (IMSS) performs a satisfaction survey among their beneficiaries. In this analysis I reveal if the IMSS has improved its satisfaction and the main satisfaction variables from the surveys. We can see in the chart that the average satisfaction is higher but to be sure we need to perform a T´Students test for identical means. The test was performed with python and the result is that the p-value is lower than .01 which means that we are sure that the satisfaction level for the 2018 was higher than for the 2017

This chart represent the correlation between the satisfaction level with the local clinic and the overall IMSS service. We can see that the satisfaction level with the local service and the overall satisfaction level is positive correlated.
